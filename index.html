Welcome to WebVulnCTF!

WVCTF or WebVulnCTF is a gamified web platform which promotes training in pentesting and web application development security in an entertaining way. Through a rewards system, the user must search for a set of flags in the vulnerable application, that correspond to frequent vulnerabilities in web applications

Show more about WVCTF

Learn web application security and have fun at the same time

WebVulnCTF allow you pentest a web application, learn security development, know the OWASP TOP 10 vulnerabilities and have fun with the gamification. You can pentest the application, get the flags and fight with your hacker friends to be the first in get the medals and trophies.

WVCTF is free and open-source and is available in this repository

Build Deploy Use


Build

To build WVCTF is necessary:

  • Java 8 (recommended jdk1.8.0_251 or latest)
  • Apache Maven 3.6.3
  • Node v12 (recommended 12.16.2) and npm 6.14.X
  • PostgreSQL (recommended v12)
  • MongoDB (recommended v4.2)

The first step is configure PostgreSQL and MongoDB. You can configure the databases and then change the /src/main/resources/application.properties (recommended) or configure the databases with the default values defined in the properties file.

The second step is build the backend with mvn clean install in [path]/wvctf

The third step is build the frontend with npm install in [path]/wvctf/src/main/java/com/julianfm/wvctf/frontend



Deploy

The first step is deploy the backend with mvn spring-boot:run in [path]/wvctf

The second step is deploy the frontend with npm start in [path]/wvctf/src/main/java/com/julianfm/wvctf/frontend



Use

Finally, you must open a browser and go to http://[host]:3000
